# Valentine's Day Card

**Description**
An interactive pixel-art Valentine's Day card built with pure HTML, CSS, and vanilla JavaScript.

## What it does
- Envelope opens to reveal the card
- Asks a question with a "No" button that runs away when you hover it
- Clicking "Yes" triggers a confetti celebration
- Floating pixel hearts you can click to pop

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript
